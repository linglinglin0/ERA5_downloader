## 📋 快速打包步骤

### 方法1: 使用自动打包脚本（推荐）

```bash
# 方案A: 源代码版
打包发布版.bat

# 方案B: Windows可执行版
打包EXE版.bat
```

### 方法2: 手动打包

#### 源代码版
```bash
# 1. 创建目录
mkdir ERA5下载器-v2.1
cd ERA5下载器-v2.1

# 2. 复制文件
copy ..\ERA5download_GUI_v2.py .
copy ..\requirements.txt .
copy ..\README.md .
copy ..\新功能使用说明.md .
copy ..\断点续传快速使用指南.md .

# 3. 创建压缩包
cd ..
powershell -Command "Compress-Archive -Path ERA5下载器-v2.1 -DestinationPath ERA5下载器-v2.1-源代码版.zip"
```
