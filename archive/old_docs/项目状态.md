## 快速开始

### 安装
```bash
git clone https://github.com/linglinglin0/ERA5_downloader.git
cd ERA5_downloader
pip install uv
uv sync
```

### 运行
```bash
# 方式1: 使用UV
uv run ERA5download_GUI_v2_fixed.py

# 方式2: 使用启动脚本
启动程序_修复版.bat
```
